Simple Email Scheduler by Wisebot.io

How it works? | Source Code on Github | Install | Uninstall

This is an add-on which helps you schedule your emails by giving simple scheduling instruction in the subject line of your draft mails which will then be sent in the required time window. Please refer to the links given above to understand how this works.

The script requires access to the following Google services...
Gmail Access

  1. To check draft messages for scheduling instructions
  2. To check replies to not send message if reply is received
  3. To send mails that after the scheduled time has arrived
  4. To modify/cleanup drafts if message sending fails
Google Drive & Spreadsheets
  1. To create folder (called WISEBOT) to store your reports
  2. To write run reports
Application Data
  1. To store user preferences
Run in background
  1. To run every 30 minutes and process mails that need to be sent
